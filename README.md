# Трекер прочитанных книг

Консольное приложение на Python для ведения списка прочитанных книг.

## Что умеет

- Добавлять книгу (автор, название, оценка 1-5, дата прочтения).
- Показывать все книги.
- Считать среднюю оценку.
- Показывать статистику по авторам.
- Удалять книгу (по индексу или по паре автор + название).

## Структура проекта

- `main.py` — консольное меню и обработка пользовательского ввода.
- `tracker.py` — бизнес-логика (работа с JSON, статистика, валидации).
- `books.json` — файл хранения данных.
- `tests/test_tracker.py` — базовые модульные тесты.

## Запуск

```bash
python main.py
```

## Запуск тестов

```bash
python -m unittest discover -s tests -p "test_*.py"
```
